# How to Run

Create a new Python environment and install all requirements listed in [REQUIREMENTS.txt](REQUIREMENTS.txt).

1. Unpack `Hu-CEMNIST_valid_images.zip` and `mnist-data.tar.gz` in [../Data](../Data)
2. Algorithmically generated counterfactuals can be produced by running [experiments.py](experiments.py)
3. Evaluation and comparision of counterfactuals are implemented in [eval.ipynb](eval.ipynb)
